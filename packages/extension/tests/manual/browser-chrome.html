Docent Manual Test — Browser Chrome Interactions

Instructions

These 9 tests involve browser chrome interactions that cannot be automated with Playwright (CDP cannot trigger browser-level shortcuts or interact with native widgets). All other capture tests are automated (48 tests).

Start recording in Docent, perform each test, commit a step after each.

Test 1: Bookmark Bar Click

Click a bookmark in the bookmark bar.

Expected: 1 navigate(auto_bookmark).

Test 2: Right-Click → Open Link in New Tab

Right-click the link below and choose "Open link in new tab" from the context menu.

Expected: 1 right_click on the link + 1 navigate(link) on the new tab. The right_click is the user's in-page action. The navigate(link) is the proxy for the context menu selection. No context_open (the new tab has an openerTabId so it's suppressed).

Right-click me → Open in new tab

Test 3: Ctrl+L (Focus Address Bar)

Press Ctrl+L to focus the address bar, then press Escape to cancel.

Expected: Nothing captured.

Test 4: Arrow Keys in Native Select

Click the select to open it, press ↓ twice, press Enter to confirm.

Expected: 1 click + 1 select.

Known limitation: Arrow keys and Enter inside native select widgets are not capturable (browser internal widget layer).

Test 5: Tab Key with Body Focused

Click on empty space (page background), then press Tab.

Expected: 1 key(Tab) with element body.

Test 6: Browser Back Button

Navigate somewhere first (e.g. use Test 1), then click the browser's Back button.

Expected: 1 navigate(back_forward).

Test 7: Ctrl+T (Open New Tab)

Press Ctrl+T to open a new tab. Submit the step from the Docent side panel (it persists across tabs).

Expected: 1 context_open (proxy for Ctrl+T). The auto-switch to the new tab is suppressed (side-effect of the open).

Cannot be automated — CDP does not trigger browser-level shortcuts.

Test 8: Ctrl+W (Close Tab)

Open a new tab (Ctrl+T), switch to it, then press Ctrl+W to close it.

Expected: 1 context_close (proxy for Ctrl+W). The auto-switch back to the previous tab is suppressed (side-effect of the close).

Cannot be automated — CDP does not trigger browser-level shortcuts.

Test 9: Click Tab to Switch

Have two tabs open. Click the other tab to switch to it.

Expected: 1 context_switch.

Cannot be automated — tab clicking is browser chrome.
