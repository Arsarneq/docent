# Reference Sync Server

A runnable reference implementation of the Docent sync protocol
(`docs/sync-protocol.md`), provided as a faithful example for adopters building
their own compatible backend and as a protocol-accurate target for the Docent
team's manual end-to-end testing.

This is a repository and testing artifact only — it is never part of a product
release.

For the full documentation (how to run it, the default port, the storage model,
optional Bearer auth, the optional conditional-write behavior, and the
non-protocol debug affordances), see
[docs/reference-sync-server.md](../../docs/reference-sync-server.md).
